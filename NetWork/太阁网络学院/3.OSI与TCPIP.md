## OSI 七层模型

|     英语     |    中文    |     巧记      |
| :----------: | :--------: | :-----------: |
| Application  |   应用层   |    Anyone     |
| Presentation |   表示层   | Password (to) |
|   Session    |   会话层   |    Secret     |
|  Transport   |   传输层   |  Tell (the)   |
|   Network    |   网络层   |      Not      |
|  Data Link   | 数据链路层 |      Do       |
|   Physical   |   物理层   |    please     |
